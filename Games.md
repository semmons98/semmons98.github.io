# Game Design

In addition to science, I have spent some time making games - both video and tabletop. I started making board games from scratch as a kid as Christmas or birthday gifts. Since then I have several work in progress tabletop games, and a video game I hope to start working on soon. This page includes descriptions of some of the games I have created as well as the rules and assets for the tabletop games.
